Recon-ng OSINT Tool Guide

🔎 Recon-ng क्या है?

Recon-ng एक powerful OSINT (Open Source Intelligence) framework है जो penetration testing और bug bounty में information gathering के लिए इस्तेमाल होता है।

यह tool internet से public information collect करता है जैसे:

  • Domains
  • Subdomains
  • Email Addresses
  • IP Address
  • Employee Names

यह tool interface में Metasploit जैसा दिखता है और modules के माध्यम से data collect करता है।

⚙️ Kali NetHunter / Kali Linux Installation

अगर tool installed नहीं है तो install करने के लिए:

sudo apt update sudo apt install recon-ng

Tool run करने के लिए:

recon-ng

🖥️ Recon-ng Interface

Tool start करने के बाद ऐसा console दिखाई देता है:

[recon-ng][default] >

Useful commands:

help modules search modules load show options run

🔎 Basic Usage (Step-by-Step)

Step 1 – Workspace Create

workspaces create target

Step 2 – Domain Add

db insert domains example.com

Step 3 – Module Load

modules load recon/domains-hosts/bing_domain_web

Step 4 – Options Check

show options

Step 5 – Module Run

run

🐞 Bug Bounty Real Example

Target domain:

example.com

Recon process:

recon-ng workspaces create example db insert domains example.com modules load recon/domains-hosts/bing_domain_web run

Possible results:

admin.example.com dev.example.com test.example.com

अब इन subdomains पर आगे security testing कर सकते हो जैसे:

  • Nmap scan
  • Nikto scan
  • SQL Injection testing
