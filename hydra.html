Hydra Password Cracking Tool Guide

Ethical Hacking Learning

🔎 Hydra क्या है?

Hydra एक powerful password cracking tool है जिसका उपयोग brute-force attack के लिए किया जाता है। यह penetration testing और ethical hacking में login authentication systems की security test करने के लिए इस्तेमाल होता है।

Hydra कई services को target कर सकता है:

  • SSH
  • FTP
  • HTTP / HTTPS
  • Telnet
  • RDP
  • MySQL
  • SMB
Hydra penetration testers और bug bounty hunters के लिए एक बहुत important tool है।

⚙️ Kali NetHunter / Kali Linux में Installation

Kali Linux और Kali NetHunter में Hydra अक्सर pre-installed आता है। अगर install नहीं है तो:

apt update apt install hydra -y

Installation check करने के लिए:

hydra -h

🧪 Basic Hydra Commands

1️⃣ SSH Password Attack

hydra -l username -P passwords.txt ssh://192.168.1.1

2️⃣ FTP Login Attack

hydra -l admin -P pass.txt ftp://192.168.1.1

3️⃣ HTTP Login Form Attack

hydra -l admin -P pass.txt 192.168.1.1 http-post-form "/login.php:user=^USER^&pass=^PASS^:F=incorrect"

4️⃣ Username + Password Wordlist Attack

hydra -L users.txt -P pass.txt ssh://192.168.1.1

5️⃣ Multi Thread Fast Attack

hydra -t 4 -l admin -P pass.txt ftp://192.168.1.1

⚡ Hydra Important Options

  • -l = Single username
  • -L = Username list
  • -p = Single password
  • -P = Password wordlist
  • -t = Threads (speed)
  • -V = Verbose output
  • -o = Output file save

🚀 Pro Tips

  • Weak passwords जल्दी crack हो जाते हैं
  • Strong password policy testing के लिए useful
  • Wordlists जैसे rockyou.txt use करें
  • Threads increase करके attack speed बढ़ा सकते हैं

❓ FAQ

Hydra किस काम आता है?

Hydra login services पर brute force attack करके weak passwords detect करने के लिए use होता है।

क्या Hydra legal है?

Hydra legal tool है लेकिन इसे बिना permission किसी system पर use करना illegal है।

क्या Hydra Kali Linux में आता है?

हाँ, Hydra Kali Linux और Kali NetHunter में अक्सर pre-installed होता है।

⚠️ Legal Warning

Hydra का उपयोग बिना permission किसी system या website पर करना illegal है।

इसे केवल ethical hacking practice labs, own servers या bug bounty scope में ही use करें।
