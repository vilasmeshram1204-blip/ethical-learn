⚡ Wifite - Automated Wireless Attack Tool

📶 Automated WiFi Hacking 🔑 WPA/WPA2/WEP ⚡ WPS Attacks

Category: 5️⃣ Wireless Attacks

📝 Description

Wifite एक ऑटोमेटेड वायरलेस अटैक टूल है जो Aircrack-ng, Reaver, और अन्य टूल्स का उपयोग करता है। यह सभी प्रक्रियाओं को ऑटोमेट करता है - WiFi नेटवर्क स्कैन करना, टारगेट चुनना, हैंडशेक कैप्चर करना, WPS अटैक करना, और पासवर्ड क्रैक करना। Wifite बिगिनर्स के लिए बहुत उपयोगी है क्योंकि यह सभी जटिल कमांड्स को एक सिंपल इंटरफेस में समेट देता है।

विशेषता: Wifite की सबसे बड़ी खासियत है इसका ऑटोमेशन - आपको अलग-अलग कमांड्स याद रखने की जरूरत नहीं। यह खुद ही सबसे अच्छा अटैक तरीका चुन लेता है (WPS पहले, फिर WPA/WPA2 हैंडशेक, फिर WEP)।

📦 Installation

Kali Linux (पहले से इंस्टॉल):

wifite --help

Ubuntu/Debian:

sudo apt update
sudo apt install wifite -y

GitHub से इंस्टॉल (नवीनतम):

git clone https://github.com/kimocoder/wifite2.git
cd wifite2
sudo python3 setup.py install

💻 Common Commands (उदाहरण सहित)

📌 Wifite का सबसे आम उपयोग:

🔹 1. बेसिक स्कैन और अटैक (सबसे आम):

sudo wifite
👉 उदाहरण: सभी आसपास के WiFi नेटवर्क स्कैन करता है। नंबर डालकर टारगेट चुनें, फिर ऑटोमेटिक अटैक शुरू हो जाता है।

🔹 2. WPA/WPA2 हैंडशेक कैप्चर और क्रैक:

sudo wifite --wpa
👉 उदाहरण: सिर्फ WPA/WPA2 नेटवर्क्स दिखाता है और हैंडशेक कैप्चर करता है।

🔹 3. WPS अटैक (Reaver):

sudo wifite --wps
👉 उदाहरण: सिर्फ WPS-सक्षम नेटवर्क्स पर अटैक करता है (PIN ब्रूटफोर्स)।

🔹 4. WEP नेटवर्क अटैक:

sudo wifite --wep
👉 उदाहरण: सिर्फ WEP नेटवर्क्स पर अटैक करता है।

🔹 5. सभी प्रकार के अटैक (डिफॉल्ट):

sudo wifite --all
👉 उदाहरण: WEP, WPA, WPS सभी पर अटैक करता है।

🔹 6. कस्टम वर्डलिस्ट के साथ क्रैकिंग:

sudo wifite --dict /usr/share/wordlists/rockyou.txt
👉 उदाहरण: rockyou.txt वर्डलिस्ट से पासवर्ड क्रैक करता है।

🔹 7. किसी स्पेसिफिक BSSID पर अटैक:

sudo wifite --bssid 00:11:22:33:44:55
👉 उदाहरण: सीधे उस BSSID वाले नेटवर्क पर अटैक शुरू करता है (बिना स्कैन के)।

🔹 8. किसी स्पेसिफिक चैनल पर स्कैन:

sudo wifite --channel 6
👉 उदाहरण: सिर्फ चैनल 6 पर स्कैन करता है (तेज स्कैन)।

🔹 9. पॉवर थ्रेशोल्ड सेट करना:

sudo wifite --power 70
👉 उदाहरण: सिर्फ 70% से अधिक सिग्नल वाले नेटवर्क दिखाता है।

🔹 10. मैक एड्रेस रैंडमाइज करना:

sudo wifite --mac
👉 उदाहरण: अटैक के दौरान रैंडम MAC एड्रेस का उपयोग करता है।

🔹 11. पिछले हैंडशेक को क्रैक करना:

sudo wifite --crack
👉 उदाहरण: पहले से सेव किए गए हैंडशेक फाइल्स को क्रैक करता है।

🔹 12. वर्बोस मोड (डिटेल्ड आउटपुट):

sudo wifite --verbose
👉 उदाहरण: स्कैन की डिटेल्ड जानकारी दिखाता है।

🎯 Wifite की अटैक प्रक्रिया:

  • Step 1: वायरलेस कार्ड को मॉनिटर मोड में डालता है
  • Step 2: आसपास के सभी नेटवर्क स्कैन करता है
  • Step 3: आपको नंबर डालकर टारगेट चुनने का विकल्प देता है
  • Step 4: स्वचालित रूप से अटैक शुरू करता है (WPS पहले, फिर हैंडशेक)
  • Step 5: हैंडशेक कैप्चर करता है (WPA/WPA2 के लिए)
  • Step 6: वर्डलिस्ट से पासवर्ड क्रैक करता है
  • Step 7: क्रैक किया हुआ पासवर्ड दिखाता है

🎯 Use Cases (उपयोग के तरीके)

  • WiFi सिक्योरिटी ऑडिट: अपने नेटवर्क की सुरक्षा जांचना (क्या WPS चालू है? पासवर्ड कितना मजबूत है?)
  • पेनिट्रेशन टेस्टिंग: वायरलेस नेटवर्क की कमजोरियां ढूँढना।
  • बिगिनर्स के लिए प्रैक्टिस: WiFi हमलों को सीखने का सबसे आसान तरीका।
  • WPS PIN टेस्टिंग: कौन से राउटर्स में WPS चालू है और PIN कमजोर है, यह जांचना।

⚙️ Key Features (मुख्य विशेषताएँ)

  • ✔️ पूरी तरह से ऑटोमेटेड (कोई जटिल कमांड याद रखने की जरूरत नहीं)
  • ✔️ WPA/WPA2, WEP, WPS (Reaver) अटैक सपोर्ट
  • ✔️ हैंडशेक कैप्चर और ऑटोमेटिक क्रैकिंग
  • ✔️ WPS PIN ब्रूटफोर्स (Reaver इंटीग्रेशन)
  • ✔️ डिअथेंटिकेशन अटैक (हैंडशेक कैप्चर के लिए)
  • ✔️ कस्टम वर्डलिस्ट सपोर्ट
  • ✔️ MAC एड्रेस रैंडमाइजेशन
  • ✔️ मल्टी-थ्रेडेड स्कैनिंग
  • ✔️ कलरफुल, यूजर-फ्रेंडली इंटरफेस
  • ✔️ सेव किए गए हैंडशेक को बाद में क्रैक कर सकता है

📌 Wifite के लिए जरूरी शर्तें:

  • ✅ वायरलेस कार्ड पैकेट इंजेक्शन सपोर्ट करता हो
  • ✅ Aircrack-ng, Reaver इंस्टॉल होना चाहिए
  • ✅ एक अच्छी वर्डलिस्ट (जैसे rockyou.txt)
  • ✅ रूट प्रिविलेज (sudo)

📌 Wifite के हॉटकी (स्कैन के दौरान):

  • Ctrl+C: स्कैन रोकना / टारगेट चुनना
  • Ctrl+Z: अटैक रोकना
  • Enter: टारगेट सेलेक्शन के बाद अटैक शुरू करना

📌 Wifite vs Aircrack-ng:

  • Wifite: ऑटोमेटेड, यूजर-फ्रेंडली, बिगिनर्स के लिए
  • Aircrack-ng: मैनुअल, ज्यादा कंट्रोल, एक्सपर्ट्स के लिए
  • अनुशंसा: Wifite से शुरुआत करें, फिर Aircrack-ng के कमांड्स सीखें।

📌 Wifite के लिए सुझाव:

  • हमेशा नवीनतम वर्शन (Wifite2) का उपयोग करें
  • पहले --wps से WPS अटैक ट्राय करें (सबसे तेज)
  • अगर WPS बंद है तो --wpa से हैंडशेक कैप्चर करें
  • अच्छी वर्डलिस्ट (rockyou.txt) का उपयोग करें
  • प्रोडक्शन नेटवर्क पर सावधानी से उपयोग करें

⚠️ Important Note (सावधानी)

Wifite एक शक्तिशाली ऑटोमेटेड टूल है जो WiFi नेटवर्क पर अटैक कर सकता है। इसका उपयोग केवल उन्हीं नेटवर्क्स पर करें जिनकी आपको लिखित अनुमति हो। बिना अनुमति के WiFi नेटवर्क स्कैन करना, हैंडशेक कैप्चर करना, या पासवर्ड क्रैक करना कानूनी अपराध है। हमेशा अपने खुद के नेटवर्क या बग बाउंटी प्रोग्राम पर ही प्रैक्टिस करें। Wifite का उपयोग सीखने और सुरक्षा ऑडिट के लिए करें, न कि अनधिकृत पहुंच के लिए।
