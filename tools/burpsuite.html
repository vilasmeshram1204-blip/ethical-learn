🎯 Burp Suite - Web Security Testing Platform

🌐 Web Proxy 🔍 Vulnerability Scanner 💼 Professional Tool

Category: 3️⃣ Web Application Attacks

📝 Description

Burp Suite दुनिया का सबसे लोकप्रिय वेब एप्लीकेशन सिक्योरिटी टेस्टिंग प्लेटफॉर्म है। यह PortSwigger द्वारा डेवलप किया जाता है। Burp Suite एक इंटरसेप्टिंग प्रॉक्सी है जो आपको वेब एप्लीकेशन और ब्राउज़र के बीच के सभी HTTP/HTTPS ट्रैफिक को कैप्चर, मॉडिफाई, और रिप्ले करने की सुविधा देता है। इसमें कई शक्तिशाली टूल्स हैं: Proxy, Intruder, Repeater, Scanner, Spider, और Decoder।

विशेषता: Burp Suite Professional (पेड) में ऑटोमेटेड स्कैनर, थ्रेडिंग, और प्रोजेक्ट मैनेजमेंट जैसे फीचर्स हैं। Burp Suite Community Edition (फ्री) में Proxy, Repeater, Intruder (लिमिटेड), और Decoder उपलब्ध हैं।

📦 Installation

Kali Linux (पहले से इंस्टॉल):

burpsuite

Windows / macOS / Linux:

https://portswigger.net/burp/releases से डाउनलोड करें और इंस्टॉल करें।

Ubuntu/Debian:

sudo apt update
sudo apt install burpsuite -y

Java आवश्यकता: Burp Suite Java में लिखा है, इसलिए Java Runtime Environment (JRE) चाहिए।

💻 Burp Suite के मुख्य टूल्स (उदाहरण सहित)

📌 Burp Suite खोलने के बाद सबसे पहले प्रॉक्सी सेट करना होता है:

Proxy → Options → Proxy Listeners → 127.0.0.1:8080 Add
Browser में Proxy: 127.0.0.1:8080 सेट करें

🔹 1. Proxy - Interceptor (ट्रैफिक कैप्चर करना)

Proxy → Intercept → Intercept is on
👉 उदाहरण: ब्राउज़र में कोई भी रिक्वेस्ट भेजें, वह Intercept हो जाएगी। आप उसे Forward, Drop, या Action → Send to Repeater/Intruder कर सकते हैं।

🔹 2. Repeater - रिक्वेस्ट को मॉडिफाई और रिप्ले करना

किसी रिक्वेस्ट पर Right Click → Send to Repeater
Repeater Tab में Request मॉडिफाई करें → Send
👉 उदाहरण: SQL Injection पेलोड डालकर Send करें और रिस्पॉन्स देखें। एक ही रिक्वेस्ट को बार-बार मॉडिफाई करके भेज सकते हैं।

🔹 3. Intruder - ऑटोमेटेड अटैक (पैरामीटर फजिंग)

रिक्वेस्ट पर Right Click → Send to Intruder
Positions Tab → पैरामीटर सेलेक्ट करें → Add §
Payloads Tab → Payload लोड करें → Start Attack
👉 उदाहरण: लॉगिन पेज पर यूजरनेम और पासवर्ड के लिए ब्रूटफोर्स अटैक करना। वर्डलिस्ट लोड करके हजारों रिक्वेस्ट भेज सकते हैं।

🔹 4. Scanner - ऑटोमेटेड वल्नरेबिलिटी स्कैन (Professional में)

Target Tab → Site Map पर Right Click → Scan
👉 उदाहरण: वेबसाइट पर SQL Injection, XSS, CSRF जैसी वल्नरेबिलिटीज ऑटोमेटिक स्कैन करता है।

🔹 5. Spider - वेबसाइट क्रॉल करना

Target Tab → Site Map पर Right Click → Spider
👉 उदाहरण: वेबसाइट के सभी लिंक्स, फॉर्म्स, JavaScript को क्रॉल करता है और Site Map में जोड़ता है।

🔹 6. Decoder - एन्कोड/डीकोड करना

Decoder Tab → Text डालें → Encode/Decode चुनें
👉 उदाहरण: Base64, URL, HTML, Hex एन्कोडिंग/डीकोडिंग कर सकते हैं। पासवर्ड हैश या टोकन डीकोड करने के लिए।

🔹 7. Comparer - रिस्पॉन्स कंपेयर करना

दो रिस्पॉन्स सेलेक्ट करें → Send to Comparer → Compare
👉 उदाहरण: दो अलग-अलग रिक्वेस्ट के रिस्पॉन्स में क्या अंतर है, यह देखने के लिए।

🔹 8. Extender - एक्सटेंशन जोड़ना

Extender Tab → BApp Store → Extensions Install
👉 उदाहरण: Turbo Intruder, Active Scan++, Logger++ जैसे एक्सटेंशन इंस्टॉल कर सकते हैं।

💻 Burp Suite के उन्नत उपयोग:

🔹 9. Match and Replace - ऑटोमेटिक रिक्वेस्ट मॉडिफाई:

Proxy → Options → Match and Replace → Add Rule
👉 उदाहरण: हर रिक्वेस्ट में User-Agent को ऑटोमेटिक बदलना या CSRF टोकन रिप्लेस करना।

🔹 10. Session Handling Rules - सेशन मैनेजमेंट:

Project Options → Sessions → Session Handling Rules
👉 उदाहरण: लॉगिन के बाद सेशन कुकी ऑटोमेटिक अपडेट करना।

🔹 11. Macros - ऑटोमेटेड एक्शन:

Project Options → Sessions → Macros → Add
👉 उदाहरण: CSRF टोकन लेने के लिए पहले एक रिक्वेस्ट भेजना, फिर टोकन के साथ दूसरी रिक्वेस्ट भेजना।

🔹 12. Scope (टारगेट सेट करना):

Target Tab → Scope → Add URL to Scope
👉 उदाहरण: सिर्फ target.com को स्कोप में रखें ताकि दूसरी वेबसाइटों का ट्रैफिक Proxy में न दिखे।

🎯 Burp Suite के मुख्य टैब्स:

  • Target: Site Map, Scope, और Target URLs की लिस्ट
  • Proxy: Intercept, HTTP History, WebSockets History
  • Intruder: पैरामीटर फजिंग और ब्रूटफोर्स अटैक
  • Repeater: रिक्वेस्ट को मैन्युअली मॉडिफाई और रिप्ले करना
  • Scanner: ऑटोमेटेड वल्नरेबिलिटी स्कैन (Professional)
  • Spider: वेबसाइट क्रॉल करना
  • Decoder: एन्कोडिंग/डीकोडिंग
  • Comparer: दो रिस्पॉन्स कंपेयर करना
  • Extender: एक्सटेंशन मैनेजमेंट
  • Logger: सभी रिक्वेस्ट/रिस्पॉन्स का पूरा लॉग

🎯 Use Cases (उपयोग के तरीके)

  • वेब एप्लीकेशन पेनिट्रेशन टेस्टिंग: SQL Injection, XSS, CSRF, IDOR, SSRF, और OWASP Top 10 वल्नरेबिलिटीज ढूँढना।
  • बग बाउंटी: टारगेट वेबसाइट पर मैनुअल और ऑटोमेटेड टेस्टिंग।
  • API टेस्टिंग: REST और GraphQL APIs की सुरक्षा टेस्टिंग।
  • रेट लिमिट बायपास: Intruder से रेट लिमिट चेक करना।
  • पैरामीटर फजिंग: Intruder से सभी पैरामीटर्स पर पेलोड्स भेजना।
  • ऑथेंटिकेशन बायपास: कुकी, टोकन, और सेशन मैनेजमेंट टेस्टिंग।

⚙️ Key Features (मुख्य विशेषताएँ)

  • ✔️ इंटरसेप्टिंग प्रॉक्सी - HTTP/HTTPS ट्रैफिक कैप्चर और मॉडिफाई
  • ✔️ Intruder - ऑटोमेटेड अटैक (ब्रूटफोर्स, फजिंग, एन्यूमरेशन)
  • ✔️ Repeater - मैनुअल रिक्वेस्ट मॉडिफाई और रिप्ले
  • ✔️ Scanner - ऑटोमेटेड वल्नरेबिलिटी स्कैन (Professional)
  • ✔️ Spider - वेबसाइट क्रॉल और मैपिंग
  • ✔️ Extender - 100+ एक्सटेंशन (BApp Store)
  • ✔️ Macros और Session Handling - ऑटोमेटेड सेशन मैनेजमेंट
  • ✔️ Match and Replace - ऑटोमेटिक रिक्वेस्ट मॉडिफाई
  • ✔️ Decoder - एन्कोडिंग/डीकोडिंग
  • ✔️ Comparer - रिस्पॉन्स कंपेयर
  • ✔️ प्रोजेक्ट फाइल्स - सारा डेटा सेव करना
  • ✔️ Team Collaboration - टीम के साथ शेयर करना (Professional)

📌 Burp Suite Community vs Professional:

  • Community (फ्री): Proxy, Repeater, Intruder (स्लो, लिमिटेड थ्रेड्स), Decoder, Comparer, Extender
  • Professional (पेड - $449/यूनिट): Scanner (ऑटोमेटेड स्कैन), Intruder (फास्ट), प्रोजेक्ट फाइल्स, Team Collaboration, CI/CD इंटीग्रेशन
  • अनुशंसा: बिगिनर्स के लिए Community Edition काफी है। प्रोफेशनल बग बाउंटी और पेनिट्रेशन टेस्टर्स Professional Edition उपयोग करते हैं।

📌 Burp Suite के लोकप्रिय एक्सटेंशन (BApp Store):

  • Turbo Intruder: बहुत तेज इंट्रूडर (रेस कंडीशन, रेट लिमिट)
  • Active Scan++: ज्यादा डीप वल्नरेबिलिटी स्कैन
  • Logger++: सभी रिक्वेस्ट/रिस्पॉन्स का डिटेल्ड लॉग
  • Autorize: IDOR (Insecure Direct Object Reference) टेस्टिंग
  • JSON Web Tokens: JWT टोकन क्रैकिंग और मॉडिफाई
  • Flow: रिक्वेस्ट/रिस्पॉन्स का टाइमलाइन व्यू

📌 Burp Suite के लिए जरूरी सावधानियाँ:

  • Intruder का उपयोग करते समय रेट लिमिट का ध्यान रखें
  • Scanner और Intruder सर्वर पर लोड बढ़ा सकते हैं
  • प्रोडक्शन सर्वर पर स्कैन करने से पहले अनुमति लें
  • Match and Replace का उपयोग सावधानी से करें

⚠️ Important Note (सावधानी)

Burp Suite एक शक्तिशाली वेब सिक्योरिटी टेस्टिंग टूल है। इसका उपयोग केवल उन्हीं वेबसाइट्स पर करें जिनकी आपको लिखित अनुमति हो। बिना अनुमति के स्कैन करना, Intruder से ब्रूटफोर्स करना, या वल्नरेबिलिटी एक्सप्लॉइट करना कानूनी अपराध है। हमेशा अपने खुद के वेब एप्लीकेशन, बग बाउंटी प्रोग्राम, या PortSwigger Web Security Academy पर ही प्रैक्टिस करें।
