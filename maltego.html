🔎 Maltego क्या है?

Maltego एक powerful OSINT tool है जो internet से publicly available data collect करता है।

  • Domain Information
  • Email Addresses
  • IP Address
  • Social Media Profiles
  • Network Connections

👉 यह data को graph (diagram) में दिखाता है जिससे connections आसानी से समझ आते हैं।

📲 Kali NetHunter Installation

apt update apt upgrade apt install maltego maltego

💻 Kali Linux Installation

maltego sudo apt update sudo apt install maltego

🚀 Basic Use (Short)

  1. Maltego start करो
  2. Login / Account बनाओ
  3. New Graph बनाओ
  4. Target add करो
  5. Run Transform
  6. Result analyze करो

🛠️ Use Kaise Kare (Detailed Practical)

  1. Maltego Open करो maltego
  2. Login / Account Create

    Free account बनाना होता है

  3. New Graph Create

    “New Graph” पर click करो

  4. Target Add करो
    • Domain (example.com)
    • Email
    • Person Name
  5. Transforms Run करो
    • Right Click → Run Transform
    • Domain → DNS info
    • Domain → IP
    • Domain → Emails
  6. Analyze करो
    • Connections graph में दिखेंगे
    • IP, Emails, Subdomains देखो
  7. Expand Data

    Double click करके और data निकालो

  8. Save Graph

    File → Save

maltego

  • Domain add करो (example.com)
