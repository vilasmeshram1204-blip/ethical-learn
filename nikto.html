Nikto Web Server Scanner Guide

🔎 Nikto क्या है?

Nikto एक powerful Open Source Web Server Scanner है जिसका उपयोग websites में security vulnerabilities खोजने के लिए किया जाता है।

यह tool penetration testing और bug bounty में बहुत useful होता है।

  • Outdated Server Software
  • Dangerous Files
  • Default Pages
  • Server Misconfiguration
  • Known Vulnerabilities

⚙️ Kali NetHunter / Kali Linux Installation

अक्सर Kali Linux में Nikto पहले से installed होता है।

Check Installation

nikto -h

Install Nikto

sudo apt update sudo apt install nikto -y

Check Version

nikto -Version

🚀 Basic Usage Commands

Basic Website Scan

nikto -h http://example.com

HTTPS Scan

nikto -h https://example.com

Specific Port Scan

nikto -h http://example.com -p 8080

Save Report

nikto -h http://example.com -o report.html

🧪 Bug Bounty Workflow

Bug bounty testing में Nikto को अक्सर Nmap के बाद use किया जाता है।

Step 1 – Port Scan

nmap -sV target.com

Step 2 – Nikto Scan

nikto -h http://target.com

Step 3 – Vulnerabilities Analyze

  • Directory Listing
  • Backup Files
  • Admin Panel Exposure
  • Outdated Server Version

⚠️ Important Notes

  • Nikto scan logs में आसानी से detect हो सकता है
  • Large websites पर scan slow हो सकता है
  • हमेशा permission लेकर ही scan करें
