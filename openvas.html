🔎 OpenVAS क्या है?

OpenVAS एक powerful open-source vulnerability scanning tool है जिसका उपयोग servers, websites और networks में security vulnerabilities detect करने के लिए किया जाता है।

यह penetration testers और bug bounty hunters द्वारा use किया जाता है।

  • Outdated software detect करता है
  • Known CVE vulnerabilities detect करता है
  • Weak SSL / TLS configuration check करता है
  • Network security issues detect करता है

⚙️ Kali NetHunter / Kali Linux में Installation

1️⃣ System Update

sudo apt update

2️⃣ OpenVAS Install

sudo apt install gvm -y

3️⃣ Setup Run करें

sudo gvm-setup

यह vulnerability database download करेगा और admin password generate करेगा।

4️⃣ Service Start करें

sudo gvm-start

🌐 Web Interface Access

Browser में OpenVAS dashboard open करें:

https://127.0.0.1:9392

Login Details:

Username: admin Password: setup के दौरान generated password

🧪 OpenVAS Basic Usage

1️⃣ Target Add करें

Configuration → Targets → New Target

example.com या 192.168.1.1

2️⃣ Scan Task Create करें

Scans → Tasks → New Task

Scan Config: Full and Fast

3️⃣ Scan Start करें

Start button दबाकर vulnerability scan run करें।

📊 Scan Report

OpenVAS report में आपको ये जानकारी मिलेगी:

  • Vulnerability Name
  • CVE ID
  • Severity Level
  • Solution / Fix

Severity Levels:

  • Low
  • Medium
  • High
  • Critical

🐞 Bug Bounty Workflow

Typical Bug Bounty Workflow:

  • Reconnaissance → Nmap / theHarvester
  • Vulnerability Scan → OpenVAS
  • Manual Testing → Burp Suite
  • Exploit Testing → SQLmap

⚠️ OpenVAS results को हमेशा manually verify करें।

⚠️ Legal Warning

OpenVAS का उपयोग केवल authorized penetration testing, bug bounty programs या lab environments में करें।

बिना permission scanning करना illegal हो सकता है।
