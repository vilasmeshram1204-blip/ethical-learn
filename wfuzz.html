🔎 WFuzz क्या है?

WFuzz एक powerful Web Application Fuzzing Tool है जिसका उपयोग वेबसाइट में hidden resources और vulnerabilities खोजने के लिए किया जाता है।

यह tool मुख्य रूप से इन चीजों को find करने में मदद करता है:

  • Hidden directories
  • Hidden files
  • GET / POST parameter fuzzing
  • Authentication brute force
  • Web vulnerabilities testing

Bug bounty और penetration testing में WFuzz बहुत उपयोगी tool है।

⚙️ Kali NetHunter / Kali Linux में Installation

अक्सर Kali Linux में WFuzz पहले से installed होता है।

अगर install नहीं है तो command run करें:

apt update
apt install wfuzz

Installation check करने के लिए:

wfuzz --help

🧪 Basic Usage Commands

1️⃣ Directory Fuzzing

wfuzz -c -z file,/usr/share/wordlists/dirb/common.txt --hc 404 http://target.com/FUZZ

यह command hidden directories खोजने के लिए use होती है।

2️⃣ File Fuzzing

wfuzz -c -z file,/usr/share/wordlists/dirb/common.txt http://target.com/FUZZ.php

3️⃣ Parameter Fuzzing

wfuzz -c -z file,wordlist.txt "http://target.com/page.php?id=FUZZ"

4️⃣ Login Brute Force

wfuzz -c -z file,users.txt -z file,passwords.txt --hs "Invalid" http://target.com/login.php?user=FUZZ&pass=FUZ2Z

📂 Wordlists Location

Kali Linux में wordlists यहाँ मिलती हैं:

/usr/share/wordlists/

Popular wordlists:

  • dirb
  • rockyou
  • seclists

🧠 Bug Bounty में WFuzz का उपयोग

  • Hidden directories खोजने के लिए
  • Hidden parameters detect करने के लिए
  • Login brute force testing
  • API fuzzing

Example:

wfuzz -c -z file,params.txt "https://site.com/api?FUZZ=test"

⚠️ Important Note

WFuzz tool को केवल legal penetration testing, lab environment, और bug bounty scope में ही use करें।

Practice के लिए labs:
