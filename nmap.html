📌 1. Nmap क्या है?

Nmap (Network Mapper) एक powerful network scanning tool है जिसका use hackers, security researchers और admins करते हैं।

  • Open ports check करने के लिए
  • Services और versions पता करने के लिए
  • Network devices ढूंढने के लिए
  • Vulnerability analysis के लिए

👉 Bug bounty और ethical hacking में यह सबसे important tool है।

⚙️ 2. Installation (Kali NetHunter)

apt update apt install nmap -y

✔️ Version check:

nmap --version

🚀 3. Basic Commands

🔹 Simple Scan

nmap 192.168.1.1

🔹 Website Scan

nmap example.com

🔹 Service + Version Scan

nmap -sV 192.168.1.1

🔹 OS Detection

nmap -O 192.168.1.1

🔹 Aggressive Scan

nmap -A 192.168.1.1

🔹 Fast Scan

nmap -F 192.168.1.1

🔹 Specific Port

nmap -p 80,443 192.168.1.1

🔹 Full Port Scan

nmap -p- 192.168.1.1

🔹 Save Output

nmap -oN output.txt 192.168.1.1

🎯 4. Bug Bounty Use

👉 Example workflow:

nmap -A example.com
  • Open ports (80, 443, 21)
  • Services (Apache, FTP)

Next steps:

  • Web testing → Burp Suite
  • SQL injection → SQLmap
  • Vulnerability scan → Nikto

⚠️ Important Notes

❗ बिना permission scanning करना illegal है

✔️ हमेशा legal labs या bug bounty scope में ही use करो

💡 Pro Tips

🔹 Aggressive scan detectable होता है

🔹 Stealth scan:

nmap -sS 192.168.1.1

🔹 Firewall bypass advanced level topic है
