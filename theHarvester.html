theHarvester Tool Guide

OSINT Information Gathering Tool

🔎 theHarvester क्या है?

theHarvester एक OSINT (Open Source Intelligence) reconnaissance tool है जिसका उपयोग किसी domain या organization के बारे में public information collect करने के लिए किया जाता है।

👉 यह मुख्य रूप से ये जानकारी collect करता है:

  • Email addresses
  • Subdomains
  • IP addresses
  • Hosts
  • Employee names

यह tool bug bounty और penetration testing में information gathering phase में बहुत useful होता है।

⚙️ Kali NetHunter / Kali Linux में Installation

अक्सर Kali Linux और NetHunter में यह पहले से install होता है।

Step 1: Package Update

apt update

Step 2: Install theHarvester

apt install theharvester

Step 3: Check Version

theHarvester -h

अगर help menu open हो जाता है तो tool successfully install हो गया।

🛠️ Basic Usage Commands

1️⃣ Domain से Emails और Subdomains निकालना

theHarvester -d example.com -b google

Explanation
-d = Target domain
-b = Data source (google, bing, yahoo, etc)

2️⃣ Multiple Sources से Data Collect करना

theHarvester -d example.com -b all

यह command सभी sources से information collect करेगी।

3️⃣ Subdomains और Hosts निकालना

theHarvester -d example.com -b bing -l 200

Explanation
-l = Results limit

4️⃣ Results को File में Save करना

theHarvester -d example.com -b google -f result.html

यह command HTML report बना देगा।

🐞 Bug Bounty में Real Use

Bug bounty hunters theHarvester का use करके hidden assets ढूंढते हैं।

Example

theHarvester -d target.com -b all

👉 इससे आपको मिल सकता है:

  • dev.target.com
  • mail.target.com
  • admin.target.com

अगर कोई hidden subdomain मिल जाए तो वहां vulnerabilities मिल सकती हैं जैसे:

  • Login panel
  • Test server
  • Development site

यहीं से bug bounty मिलने की chances बढ़ते हैं।

🔥 Pro Tips

✔ हमेशा -b all use करो ज्यादा data के लिए

✔ results को file में save करो analysis के लिए

✔ subdomains को बाद में Nmap से scan करो

Example Workflow

theHarvester -d target.com -b all nmap -sV dev.target.com
