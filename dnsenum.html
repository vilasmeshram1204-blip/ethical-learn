dnsenum Tool Guide

Information Gathering Tool

🔎 dnsenum क्या है?

dnsenum एक powerful DNS Enumeration Tool है जिसका उपयोग किसी domain की DNS information collect करने के लिए किया जाता है। यह tool bug bounty और penetration testing के reconnaissance phase में बहुत उपयोग किया जाता है।

dnsenum से आप ये जानकारी निकाल सकते हो:

  • Domain IP Address
  • Name Servers
  • MX Records
  • Subdomains
  • DNS Brute Force
  • Zone Transfer Attempt

⚙️ Kali NetHunter / Kali Linux Installation

Check करें कि dnsenum installed है या नहीं:

dnsenum --help

अगर install नहीं है:

sudo apt update sudo apt install dnsenum

🧪 Basic Usage Commands

Simple DNS Enumeration:

dnsenum example.com

Full Enumeration:

dnsenum --enum example.com

Specific DNS Server use करना:

dnsenum --dnsserver 8.8.8.8 example.com

Wordlist से Subdomain Brute Force:

dnsenum -f /usr/share/wordlists/dnsmap.txt example.com

🏆 Bug Bounty Example

मान लो target domain है:

example.com

Command run करें:

dnsenum example.com

Possible results:

mail.example.com dev.example.com api.example.com test.example.com

अब इन subdomains पर आप vulnerability testing कर सकते हो:

  • Admin Panels
  • Login Pages
  • Old Servers
  • Hidden APIs
