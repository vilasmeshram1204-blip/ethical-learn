Commix Tool Guide

🔎 Commix क्या है?

Commix (Command Injection Exploiter) एक powerful penetration testing tool है जिसका उपयोग web applications में Command Injection vulnerabilities detect और exploit करने के लिए किया जाता है।

यह tool automatically website parameters को test करता है और अगर vulnerability मिलती है तो server पर system commands execute कर सकता है।

Commix का उपयोग ethical hackers और bug bounty hunters करते हैं।

⚙️ Kali NetHunter / Kali Linux में Installation

Kali Linux में Commix अक्सर pre-installed होता है। अगर install करना हो:

apt update apt install commix

Latest version GitHub से install करने के लिए:

apt install git git clone https://github.com/commixproject/commix.git cd commix python3 commix.py

🧪 Basic Usage

1️⃣ Basic URL Scan

commix --url="http://target.com/page.php?id=1"

यह URL parameter को test करेगा कि command injection vulnerability है या नहीं।

2️⃣ POST Request Test

commix --url="http://target.com/login.php" --data="username=test&password=test"

यह login form parameters को test करेगा।

3️⃣ OS Command Execute

commix --url="http://target.com/page.php?id=1" --os-cmd="whoami"

यह server पर whoami command run करेगा।

4️⃣ Interactive Shell

commix --url="http://target.com/page.php?id=1" --os-shell

यह target server पर interactive command shell open कर सकता है।

🎯 Bug Bounty में Commix कहाँ Use होता है?

  • Command Injection Vulnerability Testing
  • Web Form Security Testing
  • Parameter Testing
  • Automated Exploitation

⚠️ Important Note

Commix tool का उपयोग केवल legal penetration testing और permission वाले targets पर ही करें। Unauthorized hacking illegal है।
