DNSRecon Tool Guide

DNSRecon Tool Guide

DNS Enumeration Tool for Ethical Hackers

🔎 DNSRecon क्या है?

DNSRecon एक powerful DNS enumeration tool है जिसका उपयोग किसी domain के DNS records और subdomains की जानकारी collect करने के लिए किया जाता है।

यह tool penetration testing और bug bounty में information gathering phase में बहुत useful होता है।

  • DNS Records
  • Subdomains
  • Name Servers
  • MX Records
  • Zone Transfer
  • TXT Records

⚙️ Kali NetHunter / Kali Linux Installation

अक्सर Kali Linux में DNSRecon पहले से install होता है।

Check करने के लिए:

dnsrecon -h

अगर install नहीं है तो install करें:

sudo apt update
sudo apt install dnsrecon

🛠️ Basic Commands

1. Basic DNS Scan

dnsrecon -d example.com

यह command collect करेगी:

  • A Records
  • NS Records
  • MX Records
  • SOA Records

2. Subdomain Enumeration

dnsrecon -d example.com -t brt

यह brute force करके hidden subdomains खोजेगा।

3. Zone Transfer Test

dnsrecon -d example.com -t axfr

अगर server misconfigured हुआ तो पूरा DNS data मिल सकता है।

4. Specific DNS Server Scan

dnsrecon -d example.com -n 8.8.8.8

🧪 Bug Bounty Example

Example target:

example.com

Command:

dnsrecon -d example.com -t brt

Possible output:

dev.example.com
api.example.com
test.example.com
admin.example.com

अब tester इन subdomains को scan करेगा:

nmap dev.example.com

nikto -h http://dev.example.com

⚠️ Legal Warning

DNS scanning और reconnaissance सिर्फ authorized targets पर ही करें। Unauthorized scanning illegal हो सकता है।
