Skipfish - Web Application Security Scanner

Skipfish Web Scanner

Fast Web Application Security Scanner

🔎 Skipfish क्या है?

Skipfish एक high speed web application security scanner है जिसे Google ने develop किया है। यह tool automated तरीके से website crawl करता है और vulnerabilities detect करता है।

यह penetration testing और bug bounty में बहुत useful होता है क्योंकि यह website की structure map करके security issues find करता है।

⚡ Skipfish Features

  • Fast Web Scanner
  • Website Crawling
  • Security Vulnerability Detection
  • Interactive HTML Reports
  • Dictionary based scanning
  • Automated Security Testing

⚙️ Kali NetHunter / Kali Linux Installation

Kali में Skipfish अक्सर pre-installed होता है। अगर नहीं है तो install करें:

apt update apt install skipfish

Installation verify करने के लिए:

skipfish -h

🛠️ Basic Usage

Simple website scan:

skipfish -o result https://example.com

यह command website scan करके report folder में save करेगी।

🧪 Example Scan

skipfish -m 5 -t 10 -o scan_result http://targetwebsite.com

Parameters:

  • -m = Maximum connections
  • -t = Request timeout
  • -o = Output directory

📊 Scan Report

Scan complete होने के बाद report folder open करें:

cd scan_result xdg-open index.html

यह browser में vulnerability report open करेगा।

⚠️ Important Note

Skipfish का उपयोग केवल Legal Penetration Testing और Bug Bounty Programs में ही करें। Unauthorized scanning illegal हो सकता है।
