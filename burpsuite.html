Burp Suite

Web Application Security Testing Tool

🔎 Burp Suite क्या है?

Burp Suite एक powerful web security testing tool है जिसका उपयोग penetration testing और bug bounty में किया जाता है। यह browser और web server के बीच होने वाले HTTP और HTTPS requests को intercept करके security vulnerabilities खोजने में मदद करता है।

Burp Suite से आप ये vulnerabilities detect कर सकते हो:

  • SQL Injection
  • Cross Site Scripting (XSS)
  • Authentication bypass
  • Session vulnerabilities
  • Parameter manipulation

⚙️ Kali NetHunter / Kali Linux Installation

Check करें कि Burp Suite installed है या नहीं:

burpsuite

अगर install नहीं है तो:

sudo apt update sudo apt install burpsuite -y

Run करने के लिए:

burpsuite

🛠️ Burp Suite के Main Tools

  • Proxy – Browser requests intercept करता है
  • Repeater – Request को modify करके बार-बार send करने के लिए
  • Intruder – Automatic payload attacks और brute force testing
  • Scanner – Automatic vulnerability scanning (Pro version)

🌐 Browser Proxy Setup

Step 1: Burp Suite open करें

Step 2: Proxy → Intercept ON

Step 3: Browser proxy configure करें:

IP: 127.0.0.1 PORT: 8080

अब browser की सभी requests Burp Suite में capture होंगी।

🧪 Basic Testing Example

1. Proxy ON करें

2. Browser में website open करें:

http://testphp.vulnweb.com

3. Burp request capture करेगा

GET /login.php HTTP/1.1 Host: testphp.vulnweb.com

🎯 Bug Bounty Workflow

Proxy → Capture Request ↓ Send to Repeater ↓ Manual Testing ↓ Send to Intruder ↓ Payload Attack

⚠️ Important Warning

Burp Suite को केवल legal targets पर use करें:

  • Bug bounty programs
  • Your own lab
  • Practice environments

Unauthorized hacking करना illegal है।
